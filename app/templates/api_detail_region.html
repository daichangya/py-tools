省市区查询API

API说明

查询中国省市区三级联动数据，支持按不同层级查询。

请求方式

GET /api/region/provinces - 获取所有省份

GET /api/region/cities?province=省份名称 - 获取某省下所有城市

GET /api/region/districts?city=城市名称 - 获取某城市下所有区县

返回示例
{
    "status": "success",
    "data": [
        {"code": "110000", "name": "北京市"},
        {"code": "310000", "name": "上海市"},
        ...
    ]
}
