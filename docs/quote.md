# 每日一言工具

## 体验地址

您可以在以下地址体验此工具：[http://api.zthinker.com/](http://api.zthinker.com/)

## 功能概述

本工具提供每日一句名言警句展示功能，从本地数据库获取数据并在网页上展示。

## 使用说明

1. 访问工具页面
2. 系统会自动加载当日的名言警句
3. 名言警句将在页面中展示
4. 页面会显示名言出处和作者信息

## 数据来源

- 名言警句来自本地数据库
- 包含古今中外各类名言
- 每日自动更新展示内容

## 技术实现

- 使用Python Flask框架实现
- 从本地SQLite数据库获取数据
- 使用Jinja2模板渲染页面
