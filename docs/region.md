# 省市区查询工具

## 体验地址

您可以在以下地址体验此工具：[http://api.zthinker.com/](http://api.zthinker.com/)

## 功能概述

本工具提供行政区划代码与名称的查询功能，支持按名称或代码两种查询方式，并可获取下级区域列表和区域详情。

## 使用说明

1. 访问工具页面
2. 选择查询方式：
   - 按名称查询：输入行政区划名称
   - 按代码查询：输入行政区划代码
3. 点击查询按钮获取结果
4. 可通过下拉菜单选择省份快速查询

## API接口

### 获取下级区域列表
```
GET /api/areas?parent_code={上级区域代码}
```
返回下级区域列表

### 获取区域详细信息
```
GET /api/area_details?code={区域代码}
```
返回区域详细信息

## 数据来源

- 基于国家统计局行政区划代码
- 包含省、市、县三级行政区划
- 数据定期更新

## 技术实现

- 使用Python Flask框架实现
- 使用SQLite数据库存储行政区划数据
- 提供RESTful API接口
- 使用Jinja2模板渲染页面
